Sử dụng API

Bạn có thể sử dụng API để dự đoán:

POST /predict
Content-Type: application/json

{
    "features": [{% for i in range(n_features) %}{% if i > 0 %}, {% endif %}0.{{ i + 1 }}{% endfor %}]
}

Ví dụ sử dụng curl:

curl -X POST http://localhost:5000/predict \
    -H "Content-Type: application/json" \
    -d '{"features": [{% for i in range(n_features) %}{% if i > 0 %}, {% endif %}0.{{ i + 1 }}{% endfor %}]}'
Về Dự án

Dự án này bao gồm:

  • Tạo dữ liệu tổng hợp từ hàm make_classification của scikit-learn
  • Huấn luyện nhiều mô hình phân loại với tinh chỉnh siêu tham số
  • So sánh và lưu mô hình được đánh giá tốt nhất
  • Web API cho phép dự đoán sử dụng mô hình tốt nhất

Để huấn luyện lại mô hình hoặc thử nghiệm với các cấu hình khác, hãy chạy script train_model.py.
